# Product Teardowns

Browse a portfolio of markdown teardowns for well-known technology products.

## Included Teardowns

- Netflix
- Uber
- Airbnb
- Stripe
- OpenAI

## Included Example Data

- `data/teardown_catalog.csv` for portfolio metadata and quick comparison
- `teardowns/*.md` for the full teardown documents

## What the App Shows

- catalog metadata for each product teardown
- full markdown analysis documents
- a portfolio-friendly browsing experience for presentations or interviews

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

The app loads the catalog dataset automatically and lets you browse each teardown without any manual setup.
